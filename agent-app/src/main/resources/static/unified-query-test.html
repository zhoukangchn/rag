💡 使用示例

1. 智能模式 (推荐)

// 客户端设置Accept头，自动选择响应模式
fetch('/ai/query?message=你好', {
    headers: { 'Accept': 'application/json' }  // → 非流式响应
});

fetch('/ai/query?message=你好', {
    headers: { 'Accept': 'text/event-stream' }  // → 流式响应
});

2. 显式控制模式

// 使用stream参数强制指定模式
GET /ai/query?message=你好&stream=true   // → 强制流式
GET /ai/query?message=你好&stream=false  // → 强制非流式

// POST请求
{
    "message": "你好",
    "stream": true,    // 强制流式
    "chainType": "standard",
    "options": {}
}
